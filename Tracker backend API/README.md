# GSCM Tracker App

Tracker Database application for the Global Supply Chain Management team

Proceedure to setup the API-local test    
1.  Install SQL server
2.  Open and run the script in Tracker_App.sql file on the SQL server. Database should now be created
3.  Lunch the Tracker backend ( Possible on VS Code)
4.  Copy the connectionString and update the appsettings.json file in the project
5.  then run from the root folder.
